Command Manual

Command Manual

Windows
Ctrl+F: Search 搜索
Ctrl+G: Find Next 下一个
Shift+Ctrl+G: Find Previous 上一个
Shift+Ctrl+F: Replace 替换
Shift+Ctrl+R: Replace All 全部替换
Alt+G: Jump To Line 跳转到行

Mac
Cmd+F: Search 搜索
Cmd+G: Find Next 下一个
Shift+Cmd+G: Find Previous 上一个
Cmd+Option+F: Replace 替换
Shift+Cmd+Option+F: Replace All 全部替换
Option+G: Jump To Line 跳转到行
